# SM-TextUtils - Django Text Analyzer

SM-TextUtils is a modern web application built using Django that helps users analyze and transform text easily. It provides powerful text processing features through a clean and responsive user interface.

## Features

Remove Punctuations
Convert to UPPERCASE
Convert to LowerCase
Remove New Lines
Remove Extra Spaces
Count Characters
